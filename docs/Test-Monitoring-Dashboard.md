## Troubleshooting

### Common Issues

#### Tests Failing in CI but Passing Locally
- Check environment variables
- Verify service dependencies are running
- Review Docker network configuration

#### Trunk.io Upload Issues
- Verify `TRUNK_API_TOKEN` secret is set
- Check network connectivity
- Review JUnit XML format

#### Keploy Test Failures
- Ensure Docker services are running
- Check network connectivity between containers
- Verify API endpoints are accessible

### Debug Commands
```bash
# Validate JUnit XML locally
python scripts/validate_junit.py junit.xml

# Check Docker network
docker network ls
docker network inspect keploy-network

# View test logs
tail -f logs/trunk-monitor.log
```
