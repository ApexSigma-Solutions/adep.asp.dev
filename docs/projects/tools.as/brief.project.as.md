# Project Brief: tools.as

The `tools.as` microservice is a foundational component of the DevEnviro Ecosystem, designed to function as a shared, reliable "utility belt" for the "Society of Agents." Its primary responsibility is to provide a suite of stateless and simple stateful tools accessible via a FastAPI interface.

This service handles common, repeatable tasks—such as accessing real-time web information, performing calculations, or maintaining a simple stateful **scratchpad**—thereby allowing agents to offload standard functions and focus on their specialized objectives. The service is architected for simplicity, reliability, and easy extension, embodying the principle of composability within the broader agentic framework.
