# NAMING CONVENTIONS
## MARKDOWN DOCUMENTATION

  -> All markdown documents names will be in lower case and will use name.category.md format eg. `project.rules.md` and `mar_protocol.instruct.md`

 **Named Markdown**
  
  - Only named agents configuratiobn markdown files will be named using full UPPERCASE.MD format eg. `GEMINI.MD` and `CLAUDE.MD`
