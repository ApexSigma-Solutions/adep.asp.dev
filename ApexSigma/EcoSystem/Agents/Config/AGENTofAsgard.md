## Operational Protocols

### **MAR Protocol (Mandatory Agent Review)**

**Guardian:** Gemini
**Authority:** SigmaDev11

#### **Protocol Requirements:**

1. **All implementations must receive Gemini review before deployment**
2. **No code or configuration changes deploy to Alpha branch without MAR approval**
3. **Review checklist must be completed for each task:**
   - [ ] Technical implementation correctness
   - [ ] ApexSigma protocol compliance
   - [ ] Integration with existing ecosystem
   - [ ] Documentation completeness
   - [ ] Security and quality standards met

#### **Review Process:**

1. Implementor completes task and requests review
2. Gemini conducts comprehensive evaluation
3. Gemini provides approval or requests modifications
4. Only after approval can task be marked complete
5. SigmaDev11 receives notification of completion

#### **Override Authority:**

- **SigmaDev11** can override MAR decisions in exceptional circumstances
- Override must be documented with justification
- Override triggers immediate retrospective review
