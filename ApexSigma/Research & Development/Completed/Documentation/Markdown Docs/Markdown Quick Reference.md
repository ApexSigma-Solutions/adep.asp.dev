## Markdown Quick Reference

**Markdown is a lightweight markup language** that allows you to format text using simple symbols and syntax. It's designed to be readable in plain text while still providing rich formatting options.[^1][^2]

### Basic Syntax Elements

**Headers**
Use hash symbols (\#) to create headings of different levels :[^1]

```
# H1 - Largest heading
## H2 - Second level  
### H3 - Third level
#### H4 - Fourth level
##### H5 - Fifth level
###### H6 - Smallest heading
```

**Text Formatting**

- **Bold text**: `**bold text**` or `__bold text__`[^1]
- *Italic text*: `*italicized text*` or `_italicized text_`[^1]
- ~~Strikethrough~~: `~~text~~`[^1]
- `Inline code`: ```code```[^2]

**Lists**
Unordered lists use dashes, asterisks, or plus signs :[^2]

```
- First item
- Second item
- Third item
```

Ordered lists use numbers :[^2]

```
1. First item
2. Second item  
3. Third item
```

**Links and Images**

- Links: `[link text](https://www.example.com)`[^1]
- Images: ``[^1]


### Advanced Elements

**Tables**
Use pipes (|) and dashes (-) to create tables [^5]:

```
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Data 1   | Data 2   | Data 3   |
| Data 4   | Data 5   | Data 6   |
```

**Code Blocks**
Use triple backticks for multi-line code :[^5]

````
```
def hello_world():
    print("Hello, World!")
```
```

**Blockquotes**
Use greater than symbol (>) for quotes :[^2]
```
> This is a blockquote
> It can span multiple lines
```

**Horizontal Rules**
Create dividers with three or more dashes, asterisks, or underscores :[^8]
```
---
***
___
```

### Extended Syntax

**Task Lists**
Create checkboxes with square brackets :[^1]
```
- [x] Completed task
- [ ] Incomplete task
```

**Footnotes**
Add footnotes to your text :[^1]
```
Here's a sentence with a footnote[^1].

[^1]: This is the footnote.
```

**Superscript and Subscript**
- Superscript: `X^2^`[^1]
- Subscript: `H~2~O`[^1]

**Text Highlighting**
Highlight important text :[^1]
```
I need to highlight these ==very important words==.
```

This reference covers the most commonly used Markdown syntax elements that work across different platforms and applications.[^5][^2][^1]
